# FA01: Techincal Test

## Ringkasan

Tidak lama lagi, kalian akan mulai berpetualang untuk mencari pekerjaan yang kalian impikan. Umumnya kalian akan menemukan techincal test berupa soal competitive programming sebagai test awal untuk menentukan apakah kalian memiliki fundamental atau problem solving skill yang cukup.

Dalam test ini, kalian akan diberikan 5 soal competitive dalam batas waktu tertentu untuk menguji kemampuan problem solving dan fundamental javascript yang sudah dipelajari.

## Aturan & Kebijakan

- Waktu Pengerjaan: **120 min** (2 jam)
- Student diharapkan menjunjung tinggi INTEGRITAS. Segala bentuk ketidakjujuran meliputi peniruan, plagiarisme, pemalsuan pengerjaan akan mendapatkan tindakan tegas dari akademik

## Bobot Penilaian

- Fundamental Assessment

## Github Live Code Workflow

Dalam pengerjaan ini, kalian diminta untuk melakukan commit sebagai checkpoin pengerjaan. Jika pengerjaan 1 soal sudah selesai, segera lakukan `add-commit` dengan message relase yang jelas.

- Contoh 1: git commit -m "Soal 1 Done"
- Contoh 2: git commit -m "Soal 3-numberCruncher: Done"

Instructor juga akan meminta commit dalam waktu tertentu, pastikan kalian melakukan `add-commit` juga ketika instructor meminta melakukan commit.

## Testing

Setiap soal sudah terdapat unit testing pada folder `__tests__`. Pastikan setiap pengerjaan yang sudah dikerjakan sesuai testing yang diharapkan. Pastikan kalian sudah `npm install`, lalu kalian bisa menjalankan:

- `npm run test`: untuk menjalankan semua test
- `npm run test [nama_file]`: untuk menjalankan test berdasarkan nama file. contoh: `npm run test 1-findDuplicate`
